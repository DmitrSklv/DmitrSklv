Пользователь может ввести ID пользователя VK и свой токен от API ЯндексДиска. Для этого нужно раскомментировать соответствующие строки кода. и закоментировать строки: 
from my_token import TOKEN
from my_token import TOKEN_VK

Фотографии сохраняются в папку: Your-Photos.
Ее адрес для приведенного образца: https://disk.yandex.ru/d/QcXFe23o2OboLQ

Информация по фотографиям отображается в файле: photos_list.txt
